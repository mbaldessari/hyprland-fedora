A collection of Hyprland and related packages for Fedora:

### Hyprland compositor

* **[hyprland-git](https://github.com/hyprwm/Hyprland)** – Dynamic tiling Wayland compositor that doesn't sacrifice on its looks (git snapshots).
* **[xdg-desktop-portal-hyprland](https://github.com/hyprwm/xdg-desktop-portal-hyprland)** – xdg-desktop-portal backend for Hyprland.

### Hyprland tools

* **[hypridle](https://github.com/hyprwm/hypridle)** – Hyprland's idle daemon.
* **[hyprlock](https://github.com/hyprwm/hyprlock)** – Hyprland's GPU-accelerated screen locking utility.
* **[hyprpaper](https://github.com/hyprwm/hyprpaper)** – Blazing fast Wayland wallpaper utility with IPC controls.
* **[hyprsunset](https://github.com/hyprwm/hyprsunset)** – An application to enable a blue-light filter on Hyprland.
* **[hyprlauncher](https://github.com/hyprwm/hyprlauncher)** – A multipurpose and versatile launcher / picker for Hyprland.

### Hyprland libraries and build dependencies

* **[aquamarine](https://github.com/hyprwm/aquamarine)** – A very light Linux rendering backend library.
* **[glaze](https://github.com/stephenberry/glaze)** – Extremely fast, in memory, JSON and interface library.
* **[hyprcursor](https://github.com/hyprwm/hyprcursor)** – The Hyprland cursor format, library and utilities.
* **[hyprgraphics](https://github.com/hyprwm/hyprgraphics)** – Hyprland graphics / resource utilities.
* **[hyprland-protocols](https://github.com/hyprwm/hyprland-protocols)** – Wayland protocol extensions for Hyprland.
* **[hyprland-qt-support](https://github.com/hyprwm/hyprland-qt-support)** – A Qt6 QML style provider for hypr* apps.
* **[hyprland-qtutils](https://github.com/hyprwm/hyprland-qtutils)** – Hyprland Qt/QML utility apps.
* **[hyprland-guiutils](https://github.com/hyprwm/hyprland-guiutils)** – Hyprland utility apps.
* **[hyprlang](https://github.com/hyprwm/hyprlang)** – The official implementation library for the hypr config language.
* **[hyprtoolkit](https://github.com/hyprwm/hyprtoolkit)** – A modern C++ Wayland-native GUI toolkit.
* **[hyprutils](https://github.com/hyprwm/hyprutils)** – Hyprland utilities library used across the ecosystem.
* **[hyprwayland-scanner](https://github.com/hyprwm/hyprwayland-scanner)** – A Hyprland implementation of wayland-scanner, in and for C++.
* **[hyprwire](https://github.com/hyprwm/hyprwire)** – A fast and consistent wire protocol for IPC.
